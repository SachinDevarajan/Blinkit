# 🛒 Blinkit Sales Analytics Dashboard

## 📌 Project Overview

This project presents an end-to-end Business Intelligence solution for Blinkit, transforming raw operational data into actionable insights through data cleaning, modeling, visualization, and analytics.

Using Python for data preprocessing and Power BI for dashboard development, the project analyzes key business areas including Sales, Customers, Inventory, Delivery Performance, and Marketing Campaigns.

---

## 🎯 Objectives

* Analyze sales performance and revenue trends
* Understand customer behavior and feedback
* Monitor inventory health and product performance
* Evaluate delivery efficiency and delays
* Measure marketing campaign effectiveness
* Generate business insights for data-driven decision making

## 📂 Dataset Overview

The project utilizes multiple datasets representing different business functions:

| Dataset               | Description                     |
| --------------------- | ------------------------------- |
| Customers             | Customer information            |
| Orders                | Order details and transactions  |
| Order Items           | Product-level order information |
| Products              | Product catalog                 |
| Customer Feedback     | Ratings and feedback            |
| Delivery Performance  | Delivery metrics and delays     |
| Inventory             | Stock and inventory data        |
| Marketing Performance | Campaign performance metrics    |

---

## 🧹 Data Cleaning & Preprocessing

Performed using Python and Pandas:

* Removed duplicates
* Handled missing values
* Standardized column names
* Converted data types
* Fixed date formats
* Created derived columns
* Exported cleaned datasets for Power BI
